# Tienda Online – Frontend (HTML & CSS)

Proyecto académico de frontend que simula una tienda online donde se exponen como catálogo productos de Apple. Los enlaces de productos redirigen a páginas oficiales de Apple para compra externa.
Este proyecto no está afiliado ni representa a ninguna marca comercial. Su propósito es únicamente educativo.

El objetivo principal del proyecto fue practicar y demostrar habilidades de maquetación, diseño visual y organización de estilos utilizando únicamente HTML y CSS, sin lógica de backend ni bases de datos.

## Tecnologías utilizadas
- HTML5
- CSS3

## Características
- Interfaz visual de tienda online
- Navegación entre múltiples páginas
- Simulación de inicio de sesión mediante validación básica en frontend
- Diseño orientado a experiencia de usuario (UI)
